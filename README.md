# Whitepapers

Whitepaper repository for Walter Henrique Alves da Silva.

PDF files are the canonical human-readable versions.  
Markdown files are extraction-based, mainly for agent/search workflows.

## Series Map

This collection is one connected framework built in layers:

1. **Boundary geometry**: how discretization creates meaning and limits.
2. **Operational formalism**: how meaning is computed (`Ax = b`, differentiation/integration).
3. **Architecture**: how self-referential systems remain stable through triadic organization.
4. **Dynamics**: how persistence, dissolution, and creation generate novelty.
5. **Physical instantiation**: how these principles appear at atomic/chemical scale.
