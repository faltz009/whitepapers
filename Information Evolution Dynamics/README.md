## Summary
This paper extends the framework into a dynamics-of-novelty model.  
It argues that persistence and elimination are not enough to explain open-ended complexity, and introduces a third operation (resonance/creation) as the missing mechanism.

Core points:
- novelty is generated by interaction dynamics, not only selected after random variation,
- integration and dissolution are complemented by a creation operation,
- cellular automata are used as demonstration systems for the argument,
- the same triadic dynamic is proposed across physical, biological, and semantic scales.

## Role in the Series
This is the dynamic layer of the framework.  
It builds on earlier boundary, operational, and architectural layers to explain how genuinely new states can emerge over time.
